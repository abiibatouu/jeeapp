# JeeApp

A Java Enterprise Edition (JEE) application developed as part of Aix-Marseille Université's Master's program.

## Overview

JeeApp is a web application built using Java Enterprise Edition technologies, providing enterprise-level features and scalability.

## Requirements

- Java 8 or higher
- Apache Maven or Gradle
- Application Server (Tomcat, WildFly, or similar)

## Building

### Using Maven

```bash
git clone https://github.com/abiibatouu/jeeapp.git
cd jeeapp
mvn clean install
```

### Using Gradle

```bash
./gradlew build
```

## Deployment

Deploy the generated WAR file to your application server.

## Features

- Enterprise-grade architecture
- Request/response handling
- Database integration
- Session management

## Technologies

- **Framework**: Java Enterprise Edition (JEE)
- **Language**: Java
- **Build Tool**: Maven / Gradle
